# Descripción del proyecto

Este proyecto consiste en el desarrollo de una **galería interactiva** utilizando JavaScript, donde el usuario puede:

- Agregar nuevas tarjetas
- Eliminar tarjetas existentes
- Filtrar tarjetas por categoría
- Visualizar un contador dinámico de tarjetas
- Ver un mensaje cuando no hay elementos en la galería

El objetivo es aplicar conceptos fundamentales de JavaScript como:

- Manipulación del DOM
- Manejo de eventos
- Delegación de eventos
- Uso de estado (arrays)
- Interacción dinámica con el usuario
